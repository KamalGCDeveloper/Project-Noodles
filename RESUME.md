## Bullet Points

- Designed and developed a RESTful financial data API using **ASP.NET Core (.NET 9)** and **C#**, serving real-time market data for stablecoins, stocks, and commodities
- Engineered a **MongoDB**-backed data layer with optimized queries for high-read workloads including price history retrieval, asset tracking, and user activity analytics
- Built paginated **search and filtering** endpoints that allow clients to query financial assets by category, name, and market metrics, improving data discoverability and API usability
- Implemented **top-growth** and **most-talked-about** asset ranking endpoints by aggregating price delta and mention-frequency data from the database
- Exposed **tracked-counts** and **active-users** analytics endpoints to surface engagement metrics for monitored financial instruments
- Delivered complete **price history** endpoints enabling clients to retrieve time-series pricing data for individual assets across stablecoins, stocks, and commodities
- Containerized the full application stack using **Docker**, enabling reproducible local development environments and streamlined deployment pipelines
- Documented all API endpoints with **Swagger / OpenAPI**, providing interactive documentation that accelerates third-party integration and internal testing
- Followed RESTful design principles including proper HTTP verb usage, resource-oriented URL structure, consistent JSON response schemas, and meaningful HTTP status codes
